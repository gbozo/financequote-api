📈 FinanceQuote API

A REST API wrapper for the Perl Finance::Quote library (v1.68). Get real-time stock quotes, currency rates, and financial data from 46+ sources.

Introduction

Base URL: http://localhost:3001/api/v1

Authentication

No authentication required by default. To enable, set API_AUTH_KEYS environment variable with comma-separated keys.

# Enable authentication
API_AUTH_KEYS=key1,key2,key3

# Usage with Bearer token
curl -H "Authorization: Bearer key1" "http://localhost:3001/api/v1/quote/AAPL"

API Endpoints

GET /api/v1/health

Check API health status

NoneNo parameters required
curl http://localhost:3001/api/v1/health
GET /api/v1/methods

List all available quote methods

NoneNo parameters required
curl http://localhost:3001/api/v1/methods
GET /api/v1/quote/:symbols

Fetch stock quotes for one or more symbols

symbolsStock ticker symbol(s), comma-separated
methodQuote source (yahoojson, alphavantage, etc.). Default: yahoojson
currencyConvert to this currency (USD, EUR, etc.)
curl "http://localhost:3001/api/v1/quote/AAPL"
curl "http://localhost:3001/api/v1/quote/AAPL,GOOGL?method=yahoojson"
curl "http://localhost:3001/api/v1/quote/AAPL?currency=EUR"
GET /api/v1/currency/:from/:to

Convert between currencies

fromSource currency code (e.g., USD)
toTarget currency code (e.g., EUR)
curl http://localhost:3001/api/v1/currency/USD/EUR
GET /api/v1/fetch/:method/:symbols

Fetch quotes using a specific method

methodQuote source method (see /api/v1/methods)
symbolsStock ticker symbol(s), comma-separated
curl "http://localhost:3001/api/v1/fetch/yahoojson/AAPL,MSFT"
